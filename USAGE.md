# B 站视频文稿助手 - 使用说明

## 新功能：AI 转换字幕为文章

### 功能概述

本插件新增了 AI 转换功能，可以将 B 站视频的字幕转换为流畅的文章，帮助用户更好地理解和整理视频内容。

### 配置步骤

#### 1. 获取 API 密钥

- **OpenAI GPT**: 访问 [OpenAI 官网](https://platform.openai.com/api-keys) 获取 API 密钥
- **其他兼容服务**: 如 Azure OpenAI、Claude 等，请参考相应服务商的文档

#### 2. 配置插件设置

1. 右键点击浏览器工具栏中的插件图标
2. 选择"选项"或"设置"
3. 在设置页面中输入：
   - **API 端点**: 例如 `https://api.openai.com/v1/chat/completions`
   - **API 密钥**: 您获取的 API 访问密钥
   - **模型选择**: 选择您要使用的 AI 模型
     - GPT-3.5 Turbo (OpenAI)
     - GPT-4 (OpenAI)
     - GPT-4 Turbo (OpenAI)
     - Claude-3 Sonnet (Anthropic)
     - Claude-3 Haiku (Anthropic)
     - Qwen Turbo (阿里云)
     - Qwen Plus (阿里云)
           - 自定义模型：输入您自己的模型名称
    - **目标字数**: 设置转换后文章的目标字数（100-10000字），留空则保持原字幕长度
4. 点击"保存设置"

#### 3. 支持的 API 服务

- OpenAI GPT-3.5/GPT-4/GPT-4 Turbo
- Anthropic Claude-3
- 阿里云通义千问
- Azure OpenAI
- 其他兼容 OpenAI API 格式的服务

### 使用方法

#### 基本操作

1. 打开任意 B 站视频页面
2. 等待插件自动加载字幕
3. 在字幕显示区域找到"📝 转文章"按钮
4. 点击按钮开始转换
5. 等待 AI 处理完成（通常需要几秒到几十秒）
6. 在弹出的模态框中查看转换结果
7. 可以点击"📋 复制文章"按钮复制内容

#### 功能特点

- **智能优化**: 自动调整文字结构、语法错误、错别字
- **保持原意**: 尽量保留原文的核心内容和关键信息
- **流畅表达**: 使文章更符合正常的语言表达习惯
- **长度限制**: 自动处理长字幕，避免 API 调用超时
- **文章持久化**: 转换后的文章会自动保存，同一视频只需转换一次
- **快速查看**: 再次点击按钮可直接查看已转换的文章
- **字数控制**: 可设置目标字数，AI会按照指定长度生成文章

### 常见问题

#### Q: 转换失败怎么办？

A: 请检查以下几点：

- API 密钥是否正确配置
- API 密钥是否有足够的额度
- 网络连接是否正常
- API 服务是否可用

#### Q: 转换速度很慢？

A: 转换速度取决于：

- 字幕内容的长度
- API 服务的响应速度
- 网络连接状况
- 建议在网络良好的环境下使用

#### Q: 转换结果不理想？

A: 可以尝试：

- 检查字幕质量，确保内容完整
- 清除文章缓存后重新转换，AI 可能给出不同的结果
- 手动编辑转换后的文章

#### Q: 如何清除已转换的文章缓存？

A: 在插件设置页面的"数据管理"部分，点击"清除文章缓存"按钮即可删除所有保存的文章。

#### Q: 如何设置文章的目标字数？

A: 在插件设置页面的"目标字数"字段中输入您希望的文章长度（100-10000字），留空则保持原字幕长度。AI会尽量按照指定字数生成文章。

#### Q: 支持哪些语言？

A: 主要支持中文和英文，具体取决于您使用的 AI 模型能力。

### 注意事项

1. **API 费用**: 使用 AI 转换功能会产生 API 调用费用，请合理使用
2. **隐私保护**: 字幕内容会发送到 AI 服务进行处理，请注意内容隐私
3. **内容质量**: 转换效果取决于原字幕的质量和 AI 模型的能力
4. **网络要求**: 需要稳定的网络连接来调用 AI 服务

### 故障排除

#### 错误信息及解决方案

| 错误信息                              | 可能原因           | 解决方案                 |
| ------------------------------------- | ------------------ | ------------------------ |
| "请先在插件设置中配置 API 端点和密钥" | 未配置 API 信息    | 前往插件设置页面配置 API |
| "API 密钥无效"                        | API 密钥错误或过期 | 检查并更新 API 密钥      |
| "API 调用频率过高"                    | 请求过于频繁       | 稍等片刻后重试           |
| "字幕内容为空"                        | 视频没有字幕       | 确认视频包含字幕         |
| "网络连接错误"                        | 网络问题           | 检查网络连接             |
